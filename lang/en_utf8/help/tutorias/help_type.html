Type

In this field you must put the start date and time of tutorship.

  • Individual: Allow to set a tutorship that permit the students suscribe to an empty slot. The teacher may set the global duration and the time per student then the system calculates the number of students. The students could select the exact time to asist to the tutorship.
  • Group: Allow to set a tutorship for a group of students that asist together to the tutorship, the teacher may set the global duration to the tutorship and do not exist students limit.
  • Event: Set an event or notice that will be shown in the calendar with a description but it will not permit suscribe to it.
  • Review: Allow to set a review, the reviews are similar to the group tutorship and you only may set the globlal duration.

This field is required and may be modified in the future only if there are not students suscribed to the tutorship.
